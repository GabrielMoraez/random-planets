# GitSearcher App

## The Project

A React application for a job process.

Live: https://randomplanets.herokuapp.com/

### Pre requisites

```
$ node --version
v8.11.4

$ yarn --version
1.10.0

```

### How to Run

```
git clone https://github.com/GabrielMoraez/git-searcher
cd git-searcher/
yarn install
yarn start
```
